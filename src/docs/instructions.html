Instructions

Creating a Meme

  1. On the main screen, click the "New Meme" button or go to File > New Meme.
  2. Select the image you want to use. Image width and height must be between 400 and 600 pixels.
  3. If the image is too big you will be able to crop or resize. If either the width or height is too small you will need to choose a different image.
  4. Cropping: Drag a box around the area you want to use. Once it is the appropriate size, the crop button will be enabled. Clicking on it will show you a preview of the cropped image.
  5. Resizing: The width and height must be between 400 and 600 pixels. You can check the "contrain proportions" checkbox to make sure your image does not look stretched. If you do this, click the "Update Size" button to update the width/height, then click the "Resize" button.

Editing and Saving a Meme

  • You can add as much text as you like to both the top and bottom text areas. The size, color and font will be set for both top and bottom text areas.
  • Saving: You can save the meme by clicking the "Save" button. Use the radio buttons to select whether to save it as a JPEG or PNG image. You can also save the image by going to File > Export/Save Meme and selecting either JPEG or PNG.
  • The currently selected tab is the meme that will be saved.
